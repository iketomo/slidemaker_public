## 各ページに含める情報
各ページには以下の情報を含めてください：

### 1. content（コンテンツ）
ページに含める具体的な内容や要点。箇条書きにしつつ、データやファクト、メッセージなどは入力内容に合わせて具体的に記載する。

### 2. visualSuggestion（推奨される視覚表現）
このページの内容を最も効果的に伝えるための視覚表現を1つ提案してください。
例：
- "比較表" - 2つ以上の項目を並べて比較する場合
- "棒グラフ" / "円グラフ" / "折れ線グラフ" - データの推移や割合を示す場合
- "フロー図" / "プロセス図" - 手順やステップを示す場合
- "アイコン+数字" - 重要な数値を強調する場合
- "タイムライン" - 時系列の流れを示す場合
- "ビフォー・アフター" - 変化を対比する場合
- "ピラミッド図" / "階層図" - 構造や優先順位を示す場合
- "マインドマップ" - 関連性や全体像を示す場合
- "イラスト中心" - コンセプトやイメージを伝える場合

### 3. emphasis（強調すべきポイント）
このページで最も強調すべき要素を具体的に指示してください。
例：
- "50%増加という数字を大きく強調"
- "3つのステップを均等に並列表示"
- "課題の深刻さを赤色で警告表示"
- "タイトルを画面中央に大きく配置"
- "ビフォーとアフターの対比を明確に"

### 4. tone（メッセージの温度感）
このページで伝えたいメッセージの雰囲気・トーンを指定してください。
例：
- "希望を感じさせる" - ポジティブな未来を示す場合
- "危機感を煽る" - 問題の深刻さを訴える場合
- "客観的に提示" - 中立的にデータを示す場合
- "親しみやすく" - カジュアルに語りかける場合
- "信頼感を与える" - プロフェッショナルに説得する場合
- "ワクワクさせる" - 期待感を高める場合
- "共感を呼ぶ" - 聴衆の気持ちに寄り添う場合
